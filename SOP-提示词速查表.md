# 欧孕S级市场分析&产品开发 — SOP提示词速查表

> 每个环节的**执行提示词** + **数据来源** + **输出物**，方便团队同步、学习和更新。

---

## 流程全景

```
阶段一          阶段二              阶段三                阶段四             阶段五              阶段六
品类定位  →  开品需求(SKU/SPU) →  核心分析(竞品购买) →  定价/销售模型  →  数据采集(TOP50)  →  大盘概况
(A+)         (A1→A4)             (B4/5/6/7)          (1/2/3)          (8/9/10)           (一/二)
```

## 阶段一：品类拓展 SOP大纲 (A+)

### 环节 1-1：品类定位（5问）

**提示词**：
```
你是欧孕品牌品类负责人。请针对目标新品类，回答以下5个定位问题：

1. 类目定位：该品类是"品类拉新"还是"客群复购"？给出理由和数据支撑。
2. 类目季节：该品类的季节性如何？各季节客群占比预估多少？对开发节奏和备货周期的影响是什么？
3. 消费属性：消费频次和消费季节性如何？建议SKU数量和价格带宽度是多少？
4. 客群年龄：品牌客群月龄段分布如何？结合当前出生率下降趋势，评估风险。
5. 消费频次：品类生命周期和产品消费周期是多少？用数据化方式描述客群画像模型。

输出：5个问题的完整答案，附数据支撑。
```

**数据源**：情报通BI + 生参数据 + 达摩盘

### 环节 1-2：大盘分析

**提示词**：
```
你是运营助理。请对以下大盘板块进行数据分析，输出各板块的年度趋势报告：

采集板块：婴童大盘 / 孕产大盘 / 亲子装大盘 / 亲子鞋大盘 / 婴童尿裤大盘 / 婴童洗护大盘

数据获取方式：
- 情报通BI → 行业分析 → 行业数据 → 三级类目 → 年度时间范围
- 提取：每月销售额、计算月占比、品牌排名

输出：每个板块的透视表格 + 趋势小结（3句话以内）
```

**数据源**：情报通BI + 淘系生参

## 阶段二：开品需求 (A1→A2→A3→A4)

### 环节 2-1：SKU维度开品需求 (A1)

**提示词**：
```
你是品类负责人。请对目标品类输出SKU维度的开品需求表，包含以下所有字段：

【SKU策划区】
- 品类布局优先顺序：按1/2/3排序
- 产品消费属性&场景：日常内着/外出/送礼等（参考TOP品+AI检索）
- 切入价格带&消费客群：如189-219元 / 新生儿妈妈
- SKU名称策划方向：AI倒推热门词+好寓意（如"清荷XX系列"）
- SKU色系开发方向：AI倒推（类目数据+高点图+首图分析）
- SKU竞对设计参考：pinterest/Google Gemini/行业优秀设计
- SKU客群月龄：达摩盘/生参
- SKU消费频次：TOP品参考+AI检索（如年度2-3件）
- SKU布局组合逻辑：大SKU全系布局四季
- 包装方式：普通/礼盒

【竞对概况区】
- 竞对品牌、竞品卖点、竞品链接、竞品价格带（日常/A/S级）
- SPU维度竞品名称、SKU价格、竞品链接截图

输出：完整A1表（10+列策划区 + 7列竞对区）
```

**数据源**：AI检索 + TOP品参考 + 达摩盘 + 手搜竞品

## 阶段四：定价定级 & 销售模型 (模块1/2/3)

### 环节 4-1：定价定级 (模块1)

**提示词**：
```
你是运营组长。请按四大定价原则搭建定价表：

定价原则：
1. 整体定位中端，参考毛利、竞品、市场环境和自身情况
2. 定价确认后一般不允许破价；特殊破价需提前申请
3. A级和S级活动采用立减金额方式，确保各渠道价拉通
4. 淘客/挑客以单链接1件为主

【定价表字段】（每个SKU一行，日常/A级/A+级/S级 四个层级各一列）：
- 序列、件数、产品名称、产品定级(S/A/B)
- 款式编码(OY1-25AW-XXXXX-X)、颜色、规格
- 成本价、前台价、优惠券

【公式区】（自动计算）：
- 日常到手价 = 前台价 - 优惠券
- 日常扣点 = 到手价 × 5.5%
- 日常毛利额 = 到手价 × (1-退款率) - 扣点 - 快递费 - 成本 - 到手价×营销费比
- 日常毛利率 = 毛利额 / 到手价
- 保本ROI = 1 / 毛利率

【活动层级】：
- A级：到手价 = 前台价×0.9 - 优惠券（10%立减）
- A+级：到手价 = 前台价×0.88 - 优惠券（12%立减）
- S级：到手价 = 前台价×0.85 - 优惠券（15%立减）

【竞品对标区】：
- 对标店铺、卖点、规格、日常到手价、A/S级到手价、销量、链接

默认参数：退款率0.1-0.15、营销费比0.1、快递费2.5

输出：完整定价表，所有层级均用公式联动。
```

### 环节 4-2：销售模型 (模块3)

**提示词**：
```
你是运营组长。请建立入局销售模型：

【核心参数设定】
- 年度总目标销售额（如3,000,000）
- 目标转化率（如2.5%）
- 目标PPC（如2.5）
- 目标产品成本（如55）
- 年度总库存资金 = 含30%预留（自动计算）
- 年度总营销费用占比 = 总营销费/总目标销售额

【月度拆解】（按大盘月占比分配）：
- 月销售额 = 年度目标 × 对应月份大盘占比
- 月推广费 = 月均付费访客 × 付费访客占比 × PPC
- 月需访客 = 月销售额 / 客单价 / 转化率
- 月均单量 = 月销售额 / 客单价
- 日销售额 = 月销售额/30（同理日推广费/日需访客/日均单量）

【库存成本】：
- 产品库存成本(130%备货率) = 月销售额 × 1.3 / 30 × 备货可售天数

【经营指标】：
- 月度营销费比 = 月推广费/月销售额
- 月度毛利润 = 月销售额 × 毛利率

【两阶段假设】：
- 链接累计权重期(1-3月)：营销费比25.8%
- 链接稳定期(3月后)：营销费比13.4%

数据源：情报通BI → 行业分析 → 行业数据 → 三级类目年度 → 月占比

输出：12个月份的完整销售模型表 + 两阶段费比切换分析。
```

### 环节 4-3：产品布局-材质|目标 (模块2)

**提示词**：
```
你是运营组长。请做S/S+级品类的产品布局规划：

【核心参数】（S/S+级固定值）：
- 库存水位：1.3系数
- 营销费用：按30%平均值预估
- 库存备货可售周期：55天
- 退款率：15%

【产品信息区】（每个链接一行）：
- 链接序号、链接布局方向定位(S/A/B/类目流量)
- 产品词主打方向、类目（三级类目路径）
- 产品痛点(1-5条)、产品方案(A-E条)
- 信任背书(A类/7A/oeko-tex/CMA/CNAS)
- 产品卖点/标题热词（孕安/孕律/孕颜/孕悦/护孕）
- 内外包装、产品材质/内档材质
- 定价倍率 = 价格带平均件单价 / 预估成本
- 尺码段、预估成本、价格带平均件单价

【月度渠道销量拆解区】（12个月，9个渠道）：
渠道：天猫/京东/拼多多/抖音自播/达播/自有短视频/达人短视频/商品卡/快手
- 总月销 = SUM(各渠道)
- 销售额预估 = 总月销 × 价格带平均件单价
- 月度平均库存货值 = (总月销×1.3×均价/30) × 55天
- 营销费用预估 = 销售额预估 × 30%

输出：完整产品布局表（产品区 + 12月×9渠道矩阵 + 库存/营销预估）。
```

## 公式速查卡片

### 定价公式
```
到手价 = 前台价 × 折扣系数 - 优惠券
扣点   = 到手价 × 5.5%
毛利额 = 到手价 × (1 - 退款率) - 扣点 - 快递费 - 成本 - 到手价 × 营销费比
毛利率 = 毛利额 / 到手价
保本ROI = 1 / 毛利率
定价倍率 = 售价 / 成本
```

### 销售模型公式
```
月销售额     = 年度目标 × 月大盘占比
月推广费     = 月均付费访客 × 付费访客占比 × PPC
月需访客     = 月销售额 / 客单价 / 转化率
产品库存成本 = 月销售额 × 1.3 / 30 × 备货天数
月度营销费比 = 月推广费 / 月销售额
```

### 产品布局公式
```
全渠道总月销      = SUM(各渠道月销)
销售额预估        = 总月销 × 均价
月度库存货值      = (总月销 × 1.3 × 均价 / 30) × 备货天数
营销费用预估      = 销售额预估 × 30%
```
